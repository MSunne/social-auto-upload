# OmniDrive Admin Frontend

Internal admin console for `OmniDrive`.

## Local Run

```bash
cd /Volumes/mud/project/github/social-auto-upload/omnidrive_admin_frontend
npm install
npm run dev
```

Default local URL:

- `http://localhost:3010`

## Scope

This frontend is for internal staff only:

- operations
- finance
- support
- reviewers
- administrators
